# Persyaratan :
1. Python 3.x harus sudah terinstall di komputer Anda.
2. Pastikan Anda memiliki pustaka numpy untuk menjalankan program. Anda bisa menginstall pustaka ini menggunakan perintah:
"pip install numpy"

# Cara mengjalankan program :
Buka file program, Program akan membuka antarmuka grafis (GUI) di mana Anda dapat:
1. Memasukkan pesan yang ingin dienkripsi atau didekripsi.
2. Mengunggah file .txt yang berisi pesan.
3. Memilih metode enkripsi (Vigenere, Playfair, atau Hill Cipher).
4. Memasukkan kunci enkripsi (minimal 12 karakter).
5. Melakukan enkripsi atau dekripsi, dan menyimpan hasilnya dalam file.

# Cara menggunakan antarmuka :
1. Masukkan pesan: Anda bisa mengetik langsung di area input atau mengunggah file teks dengan menekan tombol Upload File.
2. Pilih metode cipher: Di bagian drop-down, pilih salah satu dari tiga metode cipher (Vigenere Cipher, Playfair Cipher, atau Hill Cipher).
3. Masukkan kunci: Input kunci enkripsi atau dekripsi (minimal 12 karakter).
4. Klik tombol Enkripsi/Dekripsi: Setelah semua input lengkap, tekan tombol Enkripsi untuk mengenkripsi atau Dekripsi untuk mendekripsi pesan.
5. Lihat hasil: Hasil enkripsi atau dekripsi akan ditampilkan di area hasil di bawah.
6. Simpan hasil: Untuk menyimpan hasil enkripsi atau dekripsi, tekan tombol Simpan Hasil dan pilih lokasi untuk menyimpan file.
